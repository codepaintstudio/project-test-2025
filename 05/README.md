# 第 05 题：Vue 购物车应用（30 分）

> 基于 Vue 3 + Vite 构建

## 任务描述

完成一个简单的 Vue 购物车应用。这道题考核 Vue 的核心概念，包括数据绑定、指令使用、事件处理和计算属性。

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm run dev

# 构建生产版本
pnpm run build
```
